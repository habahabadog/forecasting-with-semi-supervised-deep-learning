# FusionCast EMS Method Reproducibility Package

This repository is a clean local package for the EMS method-paper version of FusionCast.

FusionCast is framed as a semi-supervised spatiotemporal downscaling method for regional NWP-like guidance, not as a mature operational forecasting system or a standalone software-package paper.

## Current Status

- Active manuscript version: 2022 four-variable South China benchmark.
- Active source snapshot: `/tf/yjzx/new/files/20250620`.
- Variables: 2 m temperature, 10 m U wind, 10 m V wind, precipitation.
- Target field: 1 km.
- Coarse input: 5 km, generated by stride-5 sampling from the 1 km target field.
- Split: files before `20221000` are used for training/validation; files after `20221000` are the held-out test period.

This package intentionally excludes the archived 2019 three-variable material.

## Quick Smoke Test

The shape smoke test uses a random tensor and does not require the restricted art1km archive.

```bash
python scripts/model_smoke_test.py
```

Expected output shape for the default test is `(1, 4, 15, 20)` from an input tensor `(1, 12, 3, 4)`.

## Full Data Convention

The full experiment expects hourly `.npy` files named `YYYYMMDDHH.npy`, each containing a four-channel tensor:

```text
channel 0: 2 m temperature
channel 1: 10 m U wind
channel 2: 10 m V wind
channel 3: precipitation
```

The complete art1km archive is restricted by the original data provider and is not redistributed here. A small derived or synthetic demonstration subset should be added before submission if permissions allow.

## Manuscript Packaging Notes

Before using this as a public or reviewer-accessible GitHub repository, add:

- final 2022 four-variable checkpoint metadata or checkpoint files where allowed;
- `classical_baselines_2022.csv`;
- `ablation_2022.csv`;
- `event_diagnostics_2022.csv`;
- `result_manifest_2022.csv`;
- a confirmed license.

Do not add 2019 three-variable demo tensors, legacy checkpoints, or the archived December 2019 baseline results.
